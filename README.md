# Paytm Travel Flight Booking Automation

## Project Overview

This project automates the flight booking flow on the Paytm Travel website using Playwright with TypeScript. It is designed to validate the end-to-end booking journey by simulating real user interactions and verifying the application's functionality.

The framework follows the **Page Object Model (POM)** design pattern to improve code readability, maintainability, and reusability.

## Features

- Search for one-way and round-trip flights
- Select departure and destination cities
- Choose travel dates
- Select the number of travelers
- Apply filters (if applicable)
- Verify flight search results
- Navigate through the booking flow
- Reusable Page Object Model (POM) structure
- Detailed Playwright test reports

## Tech Stack

- Playwright
- TypeScript
- Node.js
- Page Object Model (POM)

## Project Goal

The objective of this project is to demonstrate end-to-end web automation using Playwright by automating a real-world flight booking scenario on the Paytm Travel website while following automation testing best practices.
